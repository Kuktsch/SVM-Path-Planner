# SVM Path Planner

Система планирования пути на основе Support Vector Machine (SVM) для обхода препятствий.

## Описание

Проект реализует алгоритм планирования пути, который использует SVM для разделения пространства на безопасные и опасные зоны. Алгоритм обучает SVM на выборке точек вокруг препятствий и использует границу решения для построения траектории от стартовой точки до цели.

## Особенности

- Кастомная реализация Kernel SVM с алгоритмом SMO (Sequential Minimal Optimization)
- RBF (Radial Basis Function) ядро для нелинейного разделения
- Автоматическая генерация обучающих выборок вокруг препятствий
- Проверка связности коридора перед построением пути
- Сглаживание пути с помощью сплайнов
- Проверка пересечения траектории с препятствиями
- Графический интерфейс на PyQt5
- Поддержка множественных паттернов классификации препятствий

## Алгоритм

1. **Генерация окружения**: Создается случайное окружение с препятствиями
2. **Создание выборки**: Генерируются точки вокруг препятствий и вдоль линии старт-цель
3. **Обучение SVM**: Обучается SVM для разделения пространства на классы
4. **Проверка связности**: Проверяется, существует ли связный коридор от старта до цели
5. **Извлечение пути**: Путь извлекается вдоль границы решения SVM
6. **Сглаживание**: Путь сглаживается с помощью сплайнов
7. **Проверка корректности пути**: Проверяется, что траектория не пересекает препятствия

## Параметры

Основные параметры, которые можно настроить:

- **C**: Параметр регуляризации SVM (по умолчанию: 100.0)
- **gamma**: Параметр RBF ядра (по умолчанию: 0.5)
- **pattern_trials**: Количество различных паттернов классификации препятствий (по умолчанию: 5)
- **grid_resolution**: Разрешение сетки для проверки связности (по умолчанию: 0.15)
- **spline_s**: Параметр сглаживания сплайна (по умолчанию: 0.2)
